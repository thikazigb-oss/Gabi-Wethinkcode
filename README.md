# Using GenAI to Support Software Development

## Overview

This repository contains my completed exercises for the **Using GenAI to Support Software Development** course.

The exercises cover a variety of topics, including:

* Code comprehension
* Documentation generation
* API documentation
* Error diagnosis
* AI-assisted testing
* Solution verification
* Design patterns
* FastAPI fundamentals
* Programming language learning
* Code readability and refactoring

## Repository Structure

Each exercise has been documented in a separate Markdown (.md) file.

## Key Learnings

Throughout this course I learned how to:

* Use AI tools to understand unfamiliar codebases
* Generate and improve technical documentation
* Analyze and diagnose software errors
* Create testing plans and verification strategies
* Learn new programming concepts using AI-assisted prompting
* Navigate technical documentation more effectively
